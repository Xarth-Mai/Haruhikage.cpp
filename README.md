# なんで春日影やったの！

One-click to play "Haruhikage"

ワンクリックで「春日影」を再生。

一键播放《春日影》。

## Build

```bash
pip install pygame pyinstaller
```

```bash
pyinstaller --onefile --add-data "haruhikage_mygo.wav;." --icon "soyorn.ico" main.py
```
